cuando ejecutos un programa node, primero se crea el main, secuencialmente se van ejecutando todas las instruciones, cuando se encuentra una tarea asincrona lo mueve a una pila que ha llamado node Apis. Las tareas que estan esta pila esperan hasta que esten listos para resolverse cuando se resuelven, pasan a las pilas de callbacks.
cuando acaba el main, se ejecutan las tareas de la cola de de callbacks y el programa acaba cuando no hay mas procesos en la cola node apis , ni en la pila de callbacks
